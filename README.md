# autonomouscabs
## Setup
### Install packages
#### Ubuntu
    `apt install cmake g++ libboost-all-dev gnat gprbuild libaunit19-dev`
#### Debian
    `apt install cmake g++ libboost-all-dev gnat gprbuild libaunit19-dev`
### Create Makefile
    `mkdir build`
    `cd build`
    `cmake ..`
### Build
    `cd build`
    `make`
